# GameCheat - Majestic RP Edition

Это восстановленная версия чита для Majestic RP (GTA 5), написанная с нуля на C++ на основе анализа оригинального приложения.

## Особенности

### Combat
- **Aimbot** - Автоцель с множеством настроек
  - Режимы: FOV, Distance, Silent
  - Выбор кости (Head, Neck, Chest)
  - Предсказание движения
  - Сглаживание
  - Silent Auto Fire
  - Random Aim
  - Visible Only
  - Damager с горячей клавишей

### Visuals
- **ESP** - Экстраполяция и визуализация
  - Скелет игроков
  - Дистанция до целей
  - Snapline (линии к целям)
  - Имена игроков
  - Оружие
  - Боксы вокруг целей
  - HP бары
  - Ammo бары
  - Radar (мини-карта)
  - Отображение админов
  - Отображение объектов и транспорта

### Misc
- **Movement** - Модификации передвижения
  - Noclip (прохождение сквозь стены)
  - Fast Run (быстрый бег)
  - Slide Run
  - Teleport to Waypoint
- **God Mode** - Бессмертие
- **Health** - Управление здоровьем
  - Reset HP
  - Reset Armour
  - Suicide
  - Apply Damage
- **Collision** - Отключение коллизий
  - Vehicle No Collision
  - Objects No Collision
- **Recoil** - Контроль отдачи
  - Recoil Share
  - Recoil Accuracy
  - Recoil Recovery
  - Animation Reload Speed

### JS Executor
- Загрузка JavaScript скриптов
- Список скриптов
- Редактирование скриптов
- Удаление скриптов
- API для взаимодействия с читом

### Logging
- Server Logger - логирование на сервер
- Event Logger - логирование событий
- Dump Server - дамп серверных данных

### Config Management
- Save/Load конфигураций
- Cloud Config - синхронизация с облаком
- Shared Key - защищенный ключ для облака
- List/Remove конфигураций

### Security
- **Anti-Cheat Bypass** - Обход античитов
  - Скрытие процесса
  - Защита памяти
  - Скрытие модулей
  - Защита потоков
  - Anti-Debug меры
  - Скремблирование сигнатур

### UI
- **DirectX 11 Overlay** - Рендеринг оверлея поверх игры
- **Neon Logo** - Красивое неоновое лого
- **Full Menu** - Полное меню со всеми настройками
- **Hotkeys** - Горячие клавиши для быстрого доступа

### Core
- **Memory Manipulation** - Чтение и запись памяти процессов
- **DLL Injection** - Инъекция DLL в целевые процессы
- **Network Client** - Связь с сервером для телеметрии и конфигурации
- **Modular Architecture** - Чистая модульная структура кода

## Требования

- Windows 10/11
- Visual Studio 2019 или новее (с поддержкой C++17)
- CMake 3.15 или новее
- DirectX 11 SDK (входит в Windows SDK)

## Сборка проекта

### GitHub Actions (автоматическая сборка)

Проект настроен для автоматической сборки через GitHub Actions при каждом push в `main` или `master`. Сборка происходит на `windows-latest` с Visual Studio 2022.

Артефакты (EXE файл) автоматически загружаются в раздел Actions -> Build GameCheat.

### Локальная сборка

#### С помощью CMake (рекомендуется)

```bash
cd GameCheat_Restored
mkdir build
cd build
cmake .. -G "Visual Studio 17 2022" -A x64
cmake --build . --config Release
```

#### С помощью Visual Studio

1. Откройте папку проекта в Visual Studio
2. Откройте `CMakeLists.txt`
3. Выберите "CMake: Configure" из меню
4. Нажмите "CMake: Build"

### Зависимости

- Windows SDK (входит в Visual Studio Build Tools)
- DirectX 11 SDK (входит в Windows SDK)
- Стандартные Windows библиотеки: `d3d11.lib`, `wininet.lib`, `ws2_32.lib`, `iphlpapi.lib`, `psapi.lib`, `shell32.lib`

## Использование

1. **Настройте config.ini:**
   ```ini
   [Server]
   host=127.0.0.1
   port=8080

   [Overlay]
   enabled=true
   color=FFFFFFFF

   [Features]
   aimbot=false
   esp=false
   wallhack=false

   [Aimbot]
   fov=90.0
   smoothness=0.5

   [General]
   target_process=rust.exe
   log_level=info
   auto_start=false
   ```

2. **Запустите целевую игру** (например, Rust.exe)

3. **Запустите чит:**
   ```bash
   .\bin\GameCheat_Restored.exe
   ```
   
   Или укажите процесс явно:
   ```bash
   .\bin\GameCheat_Restored.exe gta5.exe
   ```

## Функционал

### MemoryManager
- Поиск процессов по имени
- Чтение/запись памяти процессов
- Выделение и освобождение памяти
- Поиск паттернов в памяти
- Получение базовых адресов модулей

### DLLInjector
- Инъекция DLL в процессы
- Выгрузка DLL из процессов
- Инъекция по имени процесса или PID

### NetworkClient
- HTTP GET/POST запросы
- Асинхронные запросы
- Поддержка SSL/TLS
- Телеметрия и конфигурация

### DirectXOverlay
- Инициализация DirectX 11
- Рендеринг текста, линий, прямоугольников
- Прозрачность и смешивание цветов
- Интеграция с игровым окном

### GameCheat
- Основной цикл чита
- Управление функциями (aimbot, ESP, wallhack)
- Телеметрия на сервер
- Обновление конфигурации

## Горячие клавиши

**Примечание:** Все горячие клавиши настраиваются пользователем через систему KeyBindManager. Поддерживаются три типа биндов:
- **TOGGLE** - Нажать один раз для включения/выключения
- **HOLD** - Удерживать для активации
- **TRIGGER** - Срабатывает при каждом нажатии

**Дефолтные бинды:**
- `F1` - Открыть/закрыть меню (TOGGLE)
- `F2` - Включить/выключить aimbot (TOGGLE)
- `F3` - Включить/выключить ESP (TOGGLE)
- `F4` - Включить/выключить God Mode (TOGGLE)
- `F5` - Сохранить конфигурацию (TRIGGER)
- `F6` - Загрузить конфигурацию (TRIGGER)
- `INSERT` - Выход (TRIGGER)

**Настройка биндов:**
Бинды настраиваются в `config.ini` в секции `[KeyBinds]` или через UI меню.
